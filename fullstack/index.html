พัฒนาเว็บ

นักพัฒนาเต็มสแตก

เป็นเจ้าของ stack ทั้งหมด — จาก pixel ไปถึงฐานข้อมูล นักพัฒนา fullstack สร้างและ ship ผลิตภัณฑ์ที่สมบูรณ์ได้ด้วยตัวเอง ทำให้พวกเขาเป็นวิศวกรที่ versatile ที่สุดในวงการ

1

พื้นฐาน Frontend

เรียน HTML, CSS และ JavaScript ให้เข้มแข็งก่อนเสมอ The Odin Project คือหลักสูตรฟรีที่เน้นโปรเจกต์ พาคุณจากศูนย์ไปพร้อมทำงานด้าน frontend ได้จริง

The Odin Project →
2

React และ Next.js

เรียน React สำหรับ UI components จากนั้น Next.js สำหรับ file-based routing, server-side rendering และ API routes Next.js คือมาตรฐานสำหรับแอป React แบบ fullstack

Next.js Learn →
3

Node.js และ Express

สร้างฝั่ง server: REST API, middleware, request validation และ error handling แม้จะใช้ Next.js API routes การเข้าใจ Express ทำให้คุณเป็น fullstack developer ที่แข็งแกร่งขึ้น

Express.js →
4

ฐานข้อมูลด้วย Prisma

ใช้ PostgreSQL เป็นฐานข้อมูลและ Prisma เป็น ORM Prisma ให้ type-safe database query ใน TypeScript และจัดการ migration อัตโนมัติ ลดงาน boilerplate ได้มาก

เอกสาร Prisma →
5

ระบบ Authentication

เพิ่มระบบ login ด้วย NextAuth.js (ปัจจุบันคือ Auth.js) รองรับ OAuth providers (Google, GitHub), credentials login และ session management ด้วย boilerplate น้อยที่สุด

Auth.js →
6

Deploy — Vercel และ Railway

Deploy Next.js frontend ไปยัง Vercel (ไม่ต้องตั้งค่าเพิ่ม) และฐานข้อมูลไปยัง Railway หรือ Supabase ตั้ง environment variables แล้ว live ภายในไม่กี่นาที

Vercel →
