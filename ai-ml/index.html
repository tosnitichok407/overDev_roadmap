วิศวกร AI / ML

ออกแบบและสร้างระบบ machine learning — ตั้งแต่ฝึกโมเดลจนถึง deploy ในระดับ scale วิศวกร ML อยู่ที่จุดตัดระหว่าง software engineering และ data science

1

พื้นฐานคณิตศาสตร์

เสริมความแข็งแกร่งด้าน linear algebra (vectors, matrices, eigenvalue) และแคลคูลัส (derivatives, gradients) ซีรีส์ "Essence of Linear Algebra" ของ 3Blue1Brown บน YouTube ทำให้คณิตศาสตร์นามธรรมรู้สึกเป็นธรรมชาติ

3Blue1Brown — Linear Algebra →
2

Machine Learning — Andrew Ng

คอร์ส ML ที่ได้รับการยกย่องมากที่สุดบนอินเทอร์เน็ต ครอบคลุม supervised learning, neural network, SVM และ best practices กว่า 5 ล้านคนเคยเรียน — มีเหตุผลที่น่าเชื่อถือ

Coursera — Machine Learning →
3

Deep Learning — fast.ai

แนวทาง top-down, code-first สำหรับ deep learning คุณสร้างโมเดลที่ใช้งานได้ตั้งแต่บทเรียนแรก แล้วค่อยเข้าใจทฤษฎีทีหลัง — ตรงข้ามกับคอร์สส่วนใหญ่ ใช้งานได้จริงและฟรีทั้งหมด

fast.ai →
4

PyTorch

framework deep learning ที่ครองตลาดทั้งในด้านการวิจัยและ production เรียน tensor, autograd, neural network module และ training loop tutorial อย่างเป็นทางการมีโครงสร้างที่ดีและใช้งานได้จริง

PyTorch — เรียนพื้นฐาน →
5

HuggingFace Transformers

เข้าถึงโมเดล NLP, vision และ multimodal ระดับ state-of-the-art ด้วยโค้ดไม่กี่บรรทัด HuggingFace กลายเป็น GitHub สำหรับโมเดล AI — เข้าใจมันคือสิ่งที่ขาดไม่ได้ในยุคนี้

HuggingFace — Quick Tour →
6

Deploy โมเดล

Serve โมเดลของคุณเป็น web app หรือ API โดยใช้ HuggingFace Spaces (ฟรี) หรือ Modal การ deploy สิ่งที่ใช้งานได้จริง — แม้แค่ demo เล็กๆ — คือวิธีที่เร็วที่สุดในการเข้าใจฝั่ง production ของ ML

HuggingFace Spaces →
