นักวิทยาศาสตร์ข้อมูล

สร้างโมเดลที่เรียนรู้จากข้อมูล นักวิทยาศาสตร์ข้อมูลผสมผสานสถิติ การเขียนโปรแกรม และความรู้ domain เพื่อดึง prediction และ insight ที่ขับเคลื่อนการตัดสินใจของผลิตภัณฑ์

1

Python และพื้นฐานคณิตศาสตร์

เรียน Python syntax จากนั้นทำความคุ้นเคยกับ linear algebra (vectors, matrices) และความน่าจะเป็น ไม่ต้องมีปริญญาคณิตศาสตร์ — แต่ต้องเข้าใจว่าทำไม algorithm ถึงทำงาน ไม่ใช่แค่เรียกใช้มัน

Python — เริ่มต้น →
2

NumPy และ Pandas

NumPy จัดการ numerical array และการดำเนินการ matrix Pandas จัดการข้อมูลแบบตาราง — โหลด CSV, filter row, group, merge และ reshape Library สองตัวนี้รองรับเกือบทุกอย่างใน data science

NumPy Quickstart →
3

Scikit-learn — Machine Learning

ฝึก model แรก: linear regression, decision tree, random forest และ k-means clustering เรียน train/test split, cross-validation และวิธีประเมิน performance ของ model อย่างถูกต้อง

Scikit-learn — เริ่มต้น →
4

Data Visualization

สื่อสาร findings ได้ชัดเจนด้วย Matplotlib และ Seaborn โมเดลที่ไม่มีใครเข้าใจคือโมเดลที่ไม่มีใครใช้ Visualization สำคัญพอๆ กับความแม่นยำของโมเดลในโลกจริง

Matplotlib — บทเรียน pyplot →
5

สถิติเชิงลึก

ลึกขึ้น: การคิดแบบ Bayesian, A/B testing, p-value และ statistical power StatQuest กับ Josh Starmer อธิบายแนวคิดเหล่านี้ได้ชัดเจนกว่าคอร์สมหาวิทยาลัยส่วนใหญ่ — และฟรีบน YouTube

StatQuest — YouTube →
6

การแข่งขัน Kaggle

ฝึกกับชุดข้อมูลจริงในการแข่งขันจริง เริ่มด้วย Titanic (binary classification) หรือ House Prices (regression) อ่าน notebook ยอดนิยมเพื่อดูว่านักปฏิบัติที่มีประสบการณ์คิดอย่างไร
