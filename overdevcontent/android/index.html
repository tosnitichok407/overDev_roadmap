นักพัฒนา Android

สร้างแอปสำหรับ platform มือถือที่ใช้มากที่สุดในโลก Android ครอบคลุมอุปกรณ์กว่า 3 พันล้านเครื่อง — และด้วย Kotlin และ Jetpack Compose ประสบการณ์นักพัฒนาดีกว่าที่เคยเป็นมา

1

Kotlin

ภาษาอย่างเป็นทางการของ Android ตั้งแต่ปี 2017 Kotlin กระชับ, ปลอดภัยจาก null และ interoperable กับ Java ได้ 100% เรียน data class, extension function, coroutine และ lambda — สิ่งเหล่านี้ปรากฏตลอดในโค้ด Android

Kotlin — เริ่มต้น →
2

Jetpack Compose — UI สมัยใหม่

UI toolkit แบบ declarative ของ Google สำหรับ Android สร้าง UI ด้วย Kotlin function แทน XML layout Compose คือปัจจุบันและอนาคตของ Android UI — โปรเจกต์ใหม่ควรใช้เสมอแทนระบบ View เก่า

เอกสาร Jetpack Compose →
3

Android Studio

IDE อย่างเป็นทางการของ Android เรียน emulator (ลองขนาดอุปกรณ์ต่างๆ), Logcat สำหรับ debug, Layout Inspector สำหรับตรวจสอบ UI และ Profiler สำหรับ performance Android Studio ใช้ IntelliJ เป็นฐาน

ดาวน์โหลด Android Studio →
4

Material Design 3

design system ของ Google สำหรับ Android — และมาตรฐานที่ผู้ใช้คาดหวัง Material 3 กำหนด color system, typography, component และ motion การใช้อย่างถูกต้องทำให้แอปรู้สึกว่าเป็นของ Android จริงๆ

Material Design 3 →
5

Google Play Console

Publish แอป รัน closed testing กับผู้ทดสอบที่คุณเชิญ ดูรายงาน crash จาก Android Vitals และจัดการ store listing ทำความคุ้นเคยกับ pre-launch report ก่อนทำการ submit ครั้งแรก

Google Play Console →
💡

ข่าวดี: Android Studio ทำงานได้บน Windows, Mac และ Linux — ไม่ต้องการ hardware พิเศษ การ publish ไปยัง Play Store ต้องเสีย ค่าลงทะเบียนครั้งเดียว $25 (เทียบกับ $99 ต่อปีของ Apple) เริ่มด้วยแอปดูสภาพอากาศหรือแอปจดบันทึก เพื่อให้ได้ APK แรกที่ทำงานบนอุปกรณ์จริง
