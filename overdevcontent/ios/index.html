นักพัฒนา iOS

สร้างแอปสำหรับ iPhone และ iPad โดยใช้ toolchain สมัยใหม่ของ Apple การพัฒนา iOS มีการตั้งค่าเริ่มต้นที่ชันกว่าเว็บ — แต่ platform นี้ขัดเกลา ผู้ใช้จ่ายเงิน และมาตรฐานคุณภาพสูง

1

Swift — ภาษาของ Apple

Swift ชัดเจน ปลอดภัย และเร็ว เรียนพื้นฐาน: optionals, closures, struct เทียบกับ class, protocol และ generics หนังสือ Swift อย่างเป็นทางการครบถ้วนและมีให้ฟรีบนเว็บของ Apple

swift.org — เริ่มต้น →
2

SwiftUI — UI แบบ Declarative

สร้าง interface โดยอธิบายว่าควรมีหน้าตาอย่างไร ไม่ใช่วิธีวาดมัน "100 Days of SwiftUI" ของ Hacking with Swift คือแหล่งเรียนรู้ฟรีที่ดีที่สุด — ใช้งานได้จริง เน้นโปรเจกต์ และ Paul Hudson อัปเดตให้สม่ำเสมอ

Hacking with Swift — 100 วัน SwiftUI →
3

Xcode — IDE ของ Apple

ทำความคุ้นเคยกับ Xcode: canvas (live preview), simulator, debugger และ Instruments สำหรับ performance profiling รู้จักเครื่องมืออย่างลึกซึ้งช่วยให้ทุกอย่างที่เหลือเร็วขึ้นมาก

developer.apple.com/xcode →
4

Apple Human Interface Guidelines

HIG ของ Apple คือคัมภีร์การออกแบบสำหรับแอป iOS ผู้ตรวจสอบ App Store ใช้มัน — แอปที่ไม่ทำตามจะถูกปฏิเสธหรือได้รีวิวแย่ อ่าน sections เรื่อง navigation, typography และ control ก่อนเริ่มออกแบบ

Apple HIG →
5

App Store Connect

ส่งแอปเพื่อรับการตรวจสอบ จัดการ beta testing ด้วย TestFlight ตั้งราคาและการจัดจำหน่าย และดู downloads และรายได้ การตรวจสอบ App Store ใช้เวลา 1–3 วันสำหรับแอปใหม่

App Store Connect →
⚠️

ข้อกำหนด: การพัฒนา iOS ต้องใช้ Mac — Xcode รันได้บน macOS เท่านั้น คุณยังต้องสมัคร Apple Developer Program ที่ราคา $99 ต่อปี เพื่อ publish ไปยัง App Store หรือทดสอบบนอุปกรณ์จริง เริ่มด้วยแอปเล็กๆ — เครื่องคำนวณทิปหรือ to-do list — ก่อนสร้างสิ่งที่ซับซ้อน
