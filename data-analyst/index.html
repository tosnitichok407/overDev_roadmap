นักวิเคราะห์ข้อมูล

แปลงข้อมูลดิบเป็นการตัดสินใจทางธุรกิจ นักวิเคราะห์ข้อมูลเก็บ ทำความสะอาด และแสดงผลข้อมูลเพื่อค้นหา pattern และสื่อสาร insight ไปยังผู้ที่ไม่ใช่นักเทคนิค

1

Spreadsheet — Excel และ Google Sheets

เรียนสูตร, pivot table, VLOOKUP และกราฟ Spreadsheet คือภาษาสากลของข้อมูลธุรกิจ — นักวิเคราะห์ส่วนใหญ่ยังใช้มันทุกวัน ไม่ว่าจะก้าวหน้าไปมากแค่ไหน

คู่มือ Google Sheets →
2

SQL — Query ฐานข้อมูล

เรียน SELECT, WHERE, GROUP BY, JOIN และ subquery SQL คือทักษะที่เป็นที่ต้องการมากที่สุดสำหรับนักวิเคราะห์ข้อมูล SQLZoo มี exercise แบบ interactive ที่สอนผ่านการปฏิบัติจริง

SQLZoo →
3

Python ด้วย Pandas

ทำ analysis แบบซ้ำๆ อัตโนมัติและจัดการข้อมูลที่ใหญ่เกินไปสำหรับ spreadsheet Pandas ช่วยให้คุณ filter, merge, reshape และรวมข้อมูลด้วยโค้ดไม่กี่บรรทัด

Pandas — เริ่มต้น →
4

Data Visualization — Tableau

สร้าง dashboard แบบ interactive ที่ผู้บริหารที่ไม่ใช่นักเทคนิคเข้าใจได้จริงๆ Tableau Public ฟรีและให้คุณ publish งานออนไลน์เพื่อใส่ในพอร์ตโฟลิโอ

วิดีโอสอน Tableau →
5

สถิติ

เข้าใจ distribution, ความสัมพันธ์เทียบกับสาเหตุ, hypothesis testing และ confidence interval ไม่จำเป็นต้องมีปริญญา — คอร์สสถิติของ Khan Academy ดีมากจริงๆ และฟรี

Khan Academy — สถิติ →
6

พอร์ตโฟลิโอ — Kaggle

Publish การวิเคราะห์บน Kaggle Notebook และร่วมสนทนาในชุมชน โปรไฟล์ Kaggle สาธารณะที่มีการวิเคราะห์จริงๆ คือหลักฐานทักษะที่แข็งแกร่งสำหรับนายจ้าง

Kaggle Learn →
💡

เคล็ดลับ: หาชุดข้อมูลที่คุณสนใจจริงๆ — สถิติกีฬา, เพลง, ร้านอาหารในละแวกบ้าน — แล้วตอบคำถามจริงด้วยมัน ความอยากรู้ทำให้ analysis น่าสนใจขึ้น 10 เท่า ทั้งสำหรับการสร้างและการอ่าน
