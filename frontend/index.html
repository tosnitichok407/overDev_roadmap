นักพัฒนาฝั่งหน้า

สร้างชั้นมองเห็นของเว็บ คุณจะได้เรียนรู้วิธีที่เบราว์เซอร์แสดงผลหน้าเว็บ วิธีเขียนโค้ด UI ที่สะอาด และวิธี deploy แอปพลิเคชัน React แบบ production จริงๆ

1

HTML — โครงสร้าง

เรียน HTML แบบ semantic: headings, links, forms, lists และ media Tag แบบ semantic ช่วยให้ทั้ง accessibility และ SEO ดีขึ้น นี่คือโครงกระดูกของทุกหน้าเว็บที่คุณจะสร้างตลอดชีวิต

MDN เรียน HTML →
2

CSS — สไตล์และ Layout

เรียน box model, Flexbox และ CSS Grid เข้าใจการออกแบบ responsive ด้วย media query CSS Tricks และช่อง YouTube ของ Kevin Powell คือแหล่งเรียนรู้ฟรีที่ดีที่สุดในตอนนี้

CSS Tricks — คู่มือ Flexbox →
3

JavaScript — การโต้ตอบ

เรียนภาษาของเว็บ ครอบคลุม ES6+ syntax, DOM manipulation, events, fetch API และ async/await javascript.info คือตำราฟรีที่ครบถ้วนและชัดเจนที่สุดที่มีอยู่ในปัจจุบัน

javascript.info →
4

React — UI แบบ Component

เรียน components, props, state, hooks (useState, useEffect, useContext) และ routing ด้วย React Router เอกสาร React อย่างเป็นทางการมีตัวอย่างแบบ interactive ตลอดทั้งเล่ม

react.dev/learn →
5

TypeScript — ความปลอดภัยของ Type

เพิ่ม static types ให้กับ JavaScript TypeScript ตรวจจับบักก่อน runtime และทำให้ codebase ดูแลรักษาง่ายขึ้น เมื่อเรียนแล้วคุณจะไม่อยากกลับไปใช้ JavaScript ล้วนๆ อีก

เอกสาร TypeScript →
6

Deploy — ส่งมันออกไป

Push โปรเจกต์ของคุณไปยัง GitHub แล้ว deploy ด้วย Vercel ภายใน 2 นาที เชื่อมต่อโดเมน ตั้ง environment variables แล้วได้ URL สำหรับแสดงในพอร์ตโฟลิโอทันที
